== Description ==

Embed a Livebooklet onto your wordpress blog. 

Embed livebooklet using shortcode `[livebooklet src="{booklet embed url}" width="{booklet width}" height="{booklet height}"]`

[All Livebooklet params](http://wordpress.org/extend/plugins/livebooklet/other_notes/)

[Livebooklet plugin page](https://livebooklet.com/)

= Try other useful plugins: =

== Other Notes ==

= Livebooklet params: =
* **src** - booklet embed url
* **width** - booklet width
* **height** - booklet height

== Installation ==

1. Install and activate the plugin on the Plugins page
2. Add shortcode `[livebooklet src="{booklet embed url}" width="{booklet width}" height="{booklet height}"]` to page or post content
